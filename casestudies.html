CASE STUDY – DATA ANALYTICS

Using AirBnB’s API, we were able to extract listing data for a few major cities around the world. The dashboard below shows the insights and data visualizations we found to be most valuable. You can filter this Microsoft Power BI dashboard by year, property type, city, and neighborhood. The dashboard automatically filters the criteria selected.

There is a geographical analysis on the “Map” tab and an analytical breakdown on the “Analysis” tab. There is also a comparative analysis by city you can interact with on the “Comparison” tab, displaying cities ranked by number of listings as well as how many listings there have been over time, by city.

All of this data could help inform AirBnB about trends in major cities around the world, and as a result adjust their business strategy to be parallel with what the data inform.
